
# 📊 Trader Performance vs Market Sentiment

## 🔹 Objective
To analyze how market sentiment (Fear vs Greed) impacts trader behavior and performance.

---

## 🔹 Methodology
- Cleaned and standardized datasets
- Converted timestamps and aligned data at daily level
- Merged sentiment data with trading dataset
- Created key behavioral metrics:
  - Trade frequency
  - Trade size
  - Activity level
  - Win rate

---

## 🔹 Key Insights
- Traders show higher activity during Greed periods  
- Trade sizes increase during Greed, indicating risk-taking  
- Fear periods show cautious trading behavior  

---

## 🔹 Strategy Recommendations
- During Fear → Reduce position size and avoid overtrading  
- During Greed → Increase trade size cautiously and follow trend  

---

## 🔹 Bonus Work
- Built a simple predictive model  
- Performed trader segmentation  

---

Python, Pandas, Matplotlib, Seaborn, Scikit-learn
